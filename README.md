## Descargar Live Tex:
### Arch

```
sudo pacman -S texlive-most
```


## Compilar:
```
pdflatex main.tex
```
Esto generará ficheros .pdf .aux .log .toc

Sí la tabla de contenidos (índice) no aparece, volver a compilar
